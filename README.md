# Hanzo Homebrew Tap

Homebrew formulae for [Hanzo AI](https://hanzo.ai) tools.

## Install

```bash
brew tap hanzoai/tap
brew install hanzo-dev
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `hanzo-dev` | AI-powered development assistant |

## Usage

```bash
# Interactive mode
hanzo-dev

# Execute a prompt
hanzo-dev exec -m zen4-mini "explain this code"

# Or use the 'dev' alias
dev
```
